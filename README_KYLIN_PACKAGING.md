# 截图FTP上传工具 - 南安专用版

这是一个为银河麒麟v10系统开发的截图自动上传工具。

## 功能特点

- 支持快捷键截图（Print Screen）
- 自动上传到FTP服务器
- 适配银河麒麟v10系统
- 包含所有必要依赖

## 安装说明

1. 双击下载的DEB包
2. 或者使用命令行安装：
   ```bash
   sudo dpkg -i screenshot-ftp-uploader_kylin-v10_amd64.deb
   ```

## 使用方法

1. 从桌面快捷方式启动程序
2. 按Print Screen键进行截图
3. 截图会自动上传到FTP服务器

## 注意事项

- 本版本包含所有必要依赖，无需网络连接即可安装
- 适配银河麒麟v10系统
